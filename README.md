## Installation

### Maven

- Add the following dependency:
```xml
<dependency>
    <groupId>com.github.fabasoad</groupId>
    <artifactId>commons-lang</artifactId>
    <version>0.3.0</version>
</dependency>
```
- Add the following repository:
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```
